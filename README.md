# KM Portfolio

## 概要

このリポジトリは、私のポートフォリオを管理するためのリポジトリです。

公開ページは[こちら](https://km0912-portfolio.netlify.app/)です。

## 環境構築

### ローカル環境

1. リポジトリをクローン

2. リポジトリのディレクトリに移動

3. 依存パッケージのインストール

   ```bash
   $ npm install
   ```

4. 開発サーバーの起動

   ```bash
   $ npm run dev
   ```

5. ブラウザで[http://localhost:5173](http://localhost:5173)にアクセス

### デプロイ

1. ビルド

   ```bash
   $ npm run build
   ```

2. デプロイ

   [Netlify](https://www.netlify.com/)を使用してデプロイしています。
